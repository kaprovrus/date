📱 Настройка Telegram

Шаг 1: Создай бота

1. Напиши @BotFather в Telegram
2. Отправь команду /newbot
3. Придумай имя и username (заканчивается на "bot")
4. Скопируй полученный токен

Шаг 2: Получи Chat ID

1. Напиши своему боту любое сообщение
2. Нажми кнопку ниже, чтобы получить ID

💡 После настройки нажми "Сохранить"
